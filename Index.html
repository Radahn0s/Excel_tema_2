🎨 Stiluri de formatare
Crearea și aplicarea stilurilor individuale de formatare în Excel
💡 Ce este un stil? Un stil este un set salvat de formatări (font, mărime, culoare, borduri, format număr) care poate fi aplicat rapid pe mai multe celule pentru consistență vizuală.

📋 Pași pentru stiluri de formatare

1
Aplicare stil predefinit (Cell Styles)
  • Home → Styles → Cell Styles
  • Categorii: Good/Bad/Neutral, Data and Model, Titles, Number Format
  • Hover pentru previzualizare, click pentru aplicare
2
Creare stil nou
  • Home → Cell Styles → New Cell Style…
  • Dă un Style Name (ex: „Titlu_Meu")
  • Click Format… → setează: Number, Alignment, Font, Border, Fill, Protection
3
Modificare stil existent
  • Home → Cell Styles → Click dreapta pe stil → Modify…
  • Schimbă formatarea → OK → Toate celulele cu acel stil se actualizează automat
4
Copiere stiluri între registre
  • Home → Cell Styles → Merge Styles…
  • Alege registrul sursă → stilurile se copiază în registrul curent
5
Format Painter (Pensulă de formate)
  • Selectează celula formatată → Home → Format Painter
  • Click pe celula/zona unde aplici formatarea
  • Dublu-click pe Format Painter pentru aplicare multiplă

🔢 Tipuri de date
Categoriile de date recunoscute de Excel în celule
🔢Număr (Number)

Valori numerice: întregi, zecimale, negative. Stocate intern ca numere reale. Aliniate automat la dreapta.

1234-45.673.14
🔤Text (String)

Orice secvență de caractere. Dacă introduci un număr precedat de apostrof ('), va fi tratat ca text. Aliniat la stânga.

'0456„Prenume"
📅Dată / Oră (Date/Time)

Stocate intern ca numere (1 = 1 ian 1900). Formatate vizual ca dată sau oră. Pot fi calculate (adunare, scădere).

01/03/202514:30
✔️Logic (Boolean)

Valori TRUE sau FALSE. Rezultate din formule logice (IF, AND, OR). Aliniate centrat.

TRUEFALSE
⚠️Eroare (Error)

Excel afișează erori când formula nu poate fi calculată.

#DIV/0!#VALUE!#REF!#NAME?#N/A
📐Formulă (Formula)

Expresii care încep cu =. Calculează și returnează valori numerice, text sau logice.

=SUM(A1:A5)=A1*B1
🔍 Recunoaștere automată: Excel identifică automat tipul de date la introducere. Dacă introduci 1/3, Excel poate interpreta ca dată (1 martie). Precedă cu apostrof ' sau setează formatul celulei la Text înainte pentru a preveni conversia.
⌨️ Metode de introducere date
Liste, formulare, meniuri contextuale în Excel

📋 Validare date (Liste)

1
Creare listă dropdown
  • Selectează celula → Data → Data Validation
  • Allow: List
  • Source: valorile separate cu ; sau un interval
  • În celulă apare buton dropdown ▼
2
Mesaje de intrare / eroare
  • Fila Input Message: afișează sfat când e selectată celula
  • Fila Error Alert: configurează mesajul la date invalide

📝 Formulare de date

1
Activare formular
  • Adaugă butonul Form… în Quick Access Toolbar
  • File → Options → Quick Access Toolbar → All Commands → Form
2
Utilizare formular
  • Click pe orice celulă din tabel → butonul Form
  • Se deschide un dialog cu câmpurile din antet
  • New adaugă rând nou, Delete șterge, Find Prev/Next navigare

🖱️ Meniu contextual (Click dreapta)

•

Click dreapta pe celulă → opțiuni rapide: Insert, Delete, Format Cells (Ctrl+1), Paste Special, Insert Comment, Define Name

•
AutoComplete

Excel sugerează automat valori deja introduse în coloană. Apasă Enter pentru acceptare sau continuă să tastezi.

•
Pick From Drop-down List

Click dreapta pe celulă → Pick From Drop-down List → afișează toate valorile unice din coloana respectivă.

📋 Memoria Clipboard

1
Clipboard Office (24 elemente)
  • Home → Clipboard → butonul de extindere ↗
  • Copiezi mai multe elemente → toate se salvează
  • Click pe orice element din panoul Clipboard pentru a-l lipi
2
Paste Special (Ctrl+Alt+V)
  • Values — lipește doar valorile (fără formule)
  • Formats — lipește doar formatarea
  • Transpose — transpune rânduri↔coloane
  • Operation — adunare/scădere/înmulțire la lipire

  • Ctrl+Z — anulează ultima acțiune
  • Click pe săgeata de lângă Undo pentru a anula mai multe acțiuni deodată
  • Istoricul Undo se șterge la salvare
🔄 Automatizarea seriilor de date
AutoFill, serii personalizate și completare automată
💡 AutoFill: Excel poate extinde automat o serie de date (numere, date calendaristice, luni, zile, text numeric) dacă tragi de mânerul de umplere (pătratul mic din colțul dreapta-jos al selecției).

📋 Tipuri de serii AutoFill

1
Serie numerică
  • Scrie 1 și 2 în două celule adiacente → selectează ambele → trage mânerul în jos/dreapta
  • Excel detectează pasul și continuă: 3, 4, 5…
  • Funcționează și cu pas negativ: 10, 8 → 6, 4, 2…
2
Serii de date calendaristice
  • Introdu o dată → trage mânerul → Excel completează zilele consecutive
  • Click pe AutoFill Options pentru a alege: Fill Days / Fill Weekdays / Fill Months / Fill Years
3
Serii text + număr
  • Scrie „Produs1" → trage → Excel completează: Produs2, Produs3…
  • Luni: „Ianuarie" → trage → Februarie, Martie…
  • Zile: „Luni" → trage → Marți, Miercuri…
4
Serii personalizate (Custom Lists)
  • File → Options → Advanced → Edit Custom Lists
  • Adaugă lista ta (ex: „Nord, Sud, Est, Vest")
5
Fill Series (control avansat)
  • Home → Fill → Series…
  • Setează: Type (Linear/Growth/Date/AutoFill), Step Value, Stop Value
